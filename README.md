# Work Portfolio Website

A simple static portfolio website with smooth transitions and section reveal effects.

## Files

- `index.html` — page structure and content
- `styles.css` — layout, colors, animations, and transition effects
- `script.js` — reveal-on-scroll behavior for sections

## Run locally

1. Open `index.html` in your browser.
2. Or use a local server, for example:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
